## 2번문제

```
ZeroDivisionError : 0으로 나눌경우생기는 에러
NameError : 지역 혹은 전역 이름 공간내에서 유효하지 않은 이름 즉 정의되지 않은 변수를 호출 하였을 경우
TypeError : 적절하지 않은 타입을 넣었을때. ex 1 + '1'
IndexError : 범위 밖의 인덱스를 입력하였을때.
KeyError : 함수 등에서 키값을 잘못입력하거나 필수 키값을 입력하지 않았을때
ModuleNotFoundError : import dioafupsf 등 없는 모듈을 임포트 시도할때
ImportError : from random import daoifupsf 등 없는 것을 임포트하려할때
```
